1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. If you run `test_openai.cjs`, set `POLLINATIONS_API_KEY` as an environment variable (do not hardcode it in source files)
4. Run the app:
   `npm run dev`
